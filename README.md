------------------------------------------------------------------------------------
Flowchart
------------------------------------------------------------------------------------

- Ask the user to enter the data requested by the sales system:
- Product name
- Product price
- Product quantity

- Validate that the price is greater than 0 and ask the user to re-enter a valid value.

- Validate that the quantity is greater than 0 and ask the user to re-enter a valid value.

- If the user wants to re-enter a quantity, continue with the purchase; if the user says no, do not continue with the purchase.

- Display a summary of the entire purchase.

- End of program
