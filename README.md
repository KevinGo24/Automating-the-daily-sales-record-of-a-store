#  Flowchart
------------------------------------------------------------------------------------

1 - Ask the user to enter the data requested by the sales system:
- Product name
- Product price
- Product quantity

4 - Validate that the price is greater than 0 and ask the user to re-enter a valid value.

5 - Validate that the quantity is greater than 0 and ask the user to re-enter a valid value.

6 - If the user wants to re-enter a quantity, continue with the purchase; if the user says no, do not continue with the purchase.

7 - Display a summary of the entire purchase.

8 - End of program
